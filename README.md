# 🚀 Build With JS

A collection of mini projects built while learning **JavaScript, DOM manipulation, and frontend development**.

---

## 📌 Projects Included

### 1️⃣ Profile Card

A responsive profile card with interactive features.

**✨ Features:**

* Dark mode toggle 🌙
* Dynamic bio text switcher 🔁
* Social media links 🔗
* Clean card UI design

**🧠 What I Learned:**

* DOM selection & manipulation
* `classList` usage
* Event listeners
* Basic UI structuring

---

### 2️⃣ Smart Counter ⚡

An advanced counter with dynamic behavior and UI feedback.

**✨ Features:**

* Increase / Decrease / Reset
* Custom step input
* Limit control (max value)
* Button disabling at limits
* Dynamic color feedback (green/red)

**🧠 What I Learned:**

* State-based UI updates
* Handling user input
* Edge case handling (limits, negatives)
* Reusable logic functions

---

### 3️⃣ Hunt the Number 🎯 *(In Progress)*

A number guessing game where users try to find a hidden number.

**✨ Planned Features:**

* Random number generation
* User input comparison
* Feedback system (Too High / Too Low)
* Attempt counter
* Game reset system
